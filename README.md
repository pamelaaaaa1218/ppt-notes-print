# PPT 备注打印排版

把 PowerPoint 的**幻灯片**和**演讲者备注 / 逐字稿**排在同一页 A4 上打印，方便对着纸质稿温习、串讲。纯网页、纯本地，文件不上传任何服务器。

## 解决的痛点

要把 PPT 打印出来对着练讲，会遇到一个两难：

- **想一页 A4 多放几页幻灯片当讲义** —— PowerPoint 的「讲义」版式可以一页放 4 页、6 页，但**不显示备注**；
- **想打印带备注的版本** —— PowerPoint 的「备注页」只能**一页纸打印一页幻灯片**，几十页 PPT 打出来又厚又费纸，也不方便快速翻看。

鱼和熊掌不可兼得。这个工具补上第三种排版：

> **一页 A4 里，左边（或上面）是幻灯片画面，右边（或下面）是这一页的备注 / 逐字稿；一张纸排 2–5 组。**

既能多页一纸，又能看到逐字稿，打印出来正好用来背稿、串场。

## 功能

- 拖入 `.pptx`，自动抽取每页的演讲者备注
- **两种幻灯片画面来源**
  - **自动渲染**：只传 PPT，浏览器里直接渲染（复杂版式可能与原稿略有偏差）
  - **PDF**：同时传一份这个 PPT 导出的 PDF，画面与原稿 **100% 一致**
- **两种版式**：左图右文 / 上图下文，一键切换
- **每页 2–5 张**可选；某页备注过长时，自动缩小该行字号塞进去，保证每页张数准确
- 备注字号（小 / 中 / 大）、显示页码、隐藏无备注的页
- 一键**打印 / 导出 A4 PDF**：CSS 控制分页，每「幻灯片 + 备注」整组不会被拆到两页

## 使用方法

1. 用浏览器打开 `index.html`
2. 把 `.pptx` 拖进去
   - 想要画面精确还原：在 PowerPoint 里把这份 PPT 导出成 PDF（`文件 → 导出 → PDF`），把 **`.pptx` 和 `.pdf` 一起**拖进去 —— PDF 负责画面、PPT 负责备注，按页码自动配对
3. 调整版式 / 每页张数 / 字号
4. 点「打印 / 导出 PDF」，纸张选 **A4**，边距用默认值，关闭「页眉和页脚」

### 本地运行

纯静态网页，无需安装、无需联网。直接双击 `index.html` 即可拖入自己的文件使用。
若要用「示例文件」按钮、或让 PDF 渲染更顺畅，建议用本地服务器打开：

```bash
cd ppt-notes-print
python3 -m http.server 8000
# 浏览器访问 http://localhost:8000
```

## 隐私

所有解析、渲染都在你自己的浏览器里完成，**PPT / PDF 不会上传到任何服务器**。

## 技术说明

- 纯前端，无后端、无构建步骤
- `.pptx` 解析：用 [JSZip](https://stuk.github.io/jszip/) 解压，自行解析 OOXML —— 包括幻灯片图形、母版 / 版式占位符继承、主题配色、演讲者备注
- PDF 渲染：[pdf.js](https://mozilla.github.io/pdf.js/)
- 第三方库已放进 `lib/`，可完全离线使用

## 已知限制

「自动渲染」模式下，文字、图片、基本图形、配色都能还原，但 SmartArt、图表等复杂元素只会显示为占位框。**需要画面和原稿完全一致时，请使用「PDF」模式。**

## 目录结构

```
ppt-notes-print/
├─ index.html       界面
├─ styles.css       样式 + A4 打印规则
├─ app.js           解析 pptx / 渲染 / 排版 / 打印
├─ lib/             JSZip、pdf.js（已本地化）
└─ sample-data.js   内置示例（PPT + PDF 的 base64，供「示例文件」按钮，双击打开也能用）
```
